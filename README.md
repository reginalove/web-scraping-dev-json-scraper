# Web Scraping Dev JSON Scraper

This Python script scrapes project information from web-scraping.dev.

## Features

- Extracts project titles
- Extracts descriptions
- Extracts difficulty levels
- Extracts links
- Saves data to JSON file

## Technologies Used

- Python
- requests
- BeautifulSoup
- JSON

## Purpose

Built to practice structured web scraping and JSON data storage.
